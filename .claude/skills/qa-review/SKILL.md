# QA Review Skill

Use this skill before considering a feature complete.

## Context files

- `CLAUDE.md`
- `docs/12_ACCEPTANCE_TESTS.md`

## Checklist

- Run build/typecheck.
- Run relevant tests.
- Check tenant filtering.
- Check validation.
- Check empty/error/loading states for UI work.
- Confirm no scope creep.

Report:

- commands run
- pass/fail result
- files changed
- remaining issues
