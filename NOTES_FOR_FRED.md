# Notes for Fred

## Best coding strategy

Start with deterministic simulation first, not heavy AI. The app becomes trustworthy when the math is clear. Then add agents to explain, compare, and recommend.

## How to keep Claude Code focused

Use one build slice per session. Do not ask Claude to build “the whole app” in one shot.

## Best first MVP demo

Use the seeded Harbor Bistro data and show three decisions:

1. Raise prices by 8%.
2. Add a Friday dinner kitchen assistant.
3. Open Monday dinner only.

Those three scenarios demonstrate profit, staffing, and opening-hours intelligence.

## Main product message

“RestTwin AI helps restaurant owners test business decisions in a digital twin before risking real money.”
